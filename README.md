# My GitHub Pages Site

A clean, modern personal portfolio/website hosted on GitHub Pages.

## Quick Start

1. Fork or push this repository to GitHub
2. Go to **Settings → Pages** and select your branch
3. Your site will be live at `https://yourusername.github.io/repo-name`

## Customization

- Edit `index.html` to change content (name, projects, links)
- Edit `css/style.css` to change colors (see `:root` variables at the top)
- Add project screenshots to an `images/` folder
- Update links in the contact section
